# digital-edition-for-fun

Czy AI może automatycznie zrobić edycję cyfrową z wersji drukowanej i czy potrzebny jest do tego informatyk?

Repozytorium zawiera wynik takiego eksporymentu przygotowanego na przykładzie fragmentu edycji Acta Alexandri Regis Poloniae, Magni Ducis Lithuaniae etc. (1501-1506) ; Akta Aleksandra króla polskiego, wielkiego księcia litewskiego i t. d. (1501-1506), wydanych w 1927 roku przez Papée, Fryderyk (1856-1940), nakł. Polskiej Akademii Umiejętności, w Krakowie. W roli AI wystąpił model Germini Pro 3, 95% pracy zostało wykonane przez model.
